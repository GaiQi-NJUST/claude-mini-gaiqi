🎉 Claude Mobile 2.0

欢迎使用全新升级的 Claude Mobile！本次更新带来以下功能：

  • 左侧边栏 — 对话历史与分组管理
  • 重命名对话 — 双击标题即可编辑
  • 回退对话 — 点击重做按钮回到上一步
  • 对话分组 — 将对话归类整理
  • API 配置 — 支持 OpenAI / DeepSeek / 通义千问 等兼容端点
  • 个人信息 — 左下角个人中心
  • 内置 Skills — 预置 prompt-refine 等实用技能
